Pulse Basic v 1.3

Please support the development of Pulse, consider purchasing a Pulse Pro license.  

This gives you:

-Built in blog
-Contact form
-Advanced file manager
-Folders feature for blocks
-Multiple image galleries and two different themes
-Image and PDF file upload manager
-Re-brandable
-Direct Technical Support
-Free Upgrades
-Settings Page
-Drag & Drop, Multi-File Gallery Uploader


************** Installation **************

1. Copy the Pulse folder to the root of your domain.
2. Edit settings in includes/config.php.
